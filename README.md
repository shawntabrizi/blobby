# blobby

A simple UI to interact with [Kusama's blob chain](https://github.com/thrumdev/blobs).

- Select an account
- Transfer KSM from Kusama to Blob (and back)
- Select a namespace
- Upload your blob
- Submit your blob

Created with:

- [Polkadot React Template](https://github.com/shawntabrizi/polkadot-react-template)
